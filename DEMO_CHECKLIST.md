# Hackathon Demo Video Checklist

Follow this script closely to ensure a flawless 3-minute Hackathon pitch video.

## Preparation
- [ ] Ensure Railway deployment is 100% green and healthy.
- [ ] Open incognito window to simulate first-time user experience.
- [ ] Prepare a sample Resume JSON payload on your clipboard.
- [ ] Clear desktop clutter and disable notifications.

## Script / Flow
1. **0:00 - 0:30 (The Hook)**
   - Show the Landing Page with the premium 3D Hero image.
   - *Pitch*: "Meet CampusMate AI, the ultimate growth OS for students."
   - Click "Create Free Account" and register a test user.

2. **0:30 - 1:15 (The Resume Builder)**
   - Navigate to **Resume Builder**.
   - Show the flawlessly responsive Desktop Split Screen layout.
   - Paste the sample JSON / edit a field.
   - Click **ATS Analyze** and highlight the instant scoring.
   - Click **Export PDF** to show WeasyPrint generation.

3. **1:15 - 1:45 (The Career Roadmaps)**
   - Navigate to **Learning Roadmaps**.
   - Select the "Software Engineering" track.
   - Show the interactive Mermaid.js diagram.
   - Explain how nodes link to actionable skills and certifications.

4. **1:45 - 2:15 (The Internship Center)**
   - Navigate to **Internship Center**.
   - Bookmark an internship.
   - Highlight the "Readiness Score" feature powered by the user's resume data.

5. **2:15 - 2:45 (Admin & Architecture)**
   - Briefly switch to Admin Dashboard to show user management.
   - Show the `ARCHITECTURE.md` file to prove it's a full-stack Flask app, not just a UI mockup.

6. **2:45 - 3:00 (The Close)**
   - Return to the Dashboard.
   - *Pitch*: "CampusMate AI bridges the gap between education and employment. Thank you."
